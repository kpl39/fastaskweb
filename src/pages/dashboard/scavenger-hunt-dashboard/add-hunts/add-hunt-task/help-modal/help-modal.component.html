How to Add Tasks

Types - There are three available types of tasks to choose from, Picture, Picture with GeoLocation and GeoLocation with 3D Model.

 

Picture Tasks - The player must take a picture of the criteria in the description inside the mobile app. This type of task does not require the player to be in a specific location.

     Required Items

  • Title - Short name to identify the task.
  • Description - Tells the player what criteria must be in the picture.
  • Points - How many points the user gets for completing the task. Naturally, the harder the task the more points.

     Example

  • Title - Bus Stop Handstand
  • Description - Find a bus-stop bench with a realtor advertisement and do a handstand on the bench.
  • Points - 200

 

Picture with Geolocation Tasks - The player must take a picture of an item or landmark at a specified location. The player must be within a certain distance to the location in order to complete the task successfully. If the player submits the task and is not in range to the location they will be told to try again.

     Required Items

  • Title - Short name to identify the task.
  • Description - Give the player hints on where the location might be and tell them what needs to be in the picture.
  • Points - How many points the user gets for completing the task. Naturally, the harder the task the more points.
  • Location - Click on the map to choose the location. You can remove the marker by clicking “Remove Marker” on the map. You can also drag the marker to a different location.
  • Range - Tell the user how far away from the location they can be. A circle will be drawn around the marker with the supplied radius (in feet). If the user needs to take a picture of a large building make sure the radius is large enough that the user can take a picture from a distance.

     Example

  • Title - King Kong Ain’t Got $hi+ on Me
  • Description - Take a picture of the famous landmark that the larger than life gorilla scaled.
  • Points - 500
  • Location - Search for Empire State Building in the map’s search box. Click in the middle of the building.
  • Radius - Move the radius slider bar until the circle is completely around the building with some padding around the edges.

This Radius is too small. The player would have to be inside the building.

This radius will allow the player to take a picture of the outside of the building.
